# Renommer titre mp3

Ce script python renomme les métadonnées "titre" de vos fichiers mp3 à l'identique que les noms de fichier.

## Pour commencer

### Pré-requis

- Python
- Chemin du dossier où ce situe les fichiers mp3

## Démarrage

Lancer le script depuis l'invite de commande

## Fabriqué avec

- VS Code
